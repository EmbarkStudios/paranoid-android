pub fn init(self) where
    Subscriber<N, E>: Into<Dispatch>, 

Converts self into a Subscriber and attempts to set it as the global default subscriber in the current scope, panicking if this fails.

Panics

This method panics if a global default subscriber has already been set, or if a log logger has already been set (when the “log” feature is enabled).

log compatibility

If the “log” feature flag is enabled, this will also attempt to initialize a log compatibility layer. This allows the subscriber to consume log::Records as though they were tracing Events.
